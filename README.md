# Pixel Pattern Studio

A **Next.js + TypeScript** web app for creating and following **pixel-based crochet patterns**.

Design patterns from scratch or convert images into pixel grids, then switch to **work mode** to follow them row by row.
Tailored for **crochet tapestry**, but also suitable for **cross-stitch patterns**, as both rely on grid-based designs.

## Main Features

* Create custom pixel patterns
* Convert images into crochet-friendly designs (with adjustable color count)
* Edit patterns on an interactive grid
* Undo history
* Work mode with row tracking and progress
* Local persistence 
* Export patterns as PDF

## Getting Started

```bash
git clone https://github.com/BalaciSofia/pixel-crochet-app.git
cd pixel-crochet-app
npm install
npm run dev
```

Open: http://localhost:3000
